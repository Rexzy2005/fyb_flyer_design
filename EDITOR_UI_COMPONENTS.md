## Responsive Behavior

### Desktop (1024px+)
- Side-by-side layout
- Preview sticky on left
- Form on right
- Wide spacing
- Large buttons

### Tablet (768px - 1023px)
- Single column
- Full-width elements
- Improved touch targets
- Larger text
- Generous padding

### Mobile (< 768px)
- Full-width everything
- Stacked sections
- Touch-friendly buttons (44px minimum)
- Larger font sizes
- Simplified header
